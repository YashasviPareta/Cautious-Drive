Cautious Drive 🚗⚠️

A road-safety mobile app that alerts drivers to accident-prone zones along their route in real time, using crowdsourced hazard reporting to keep the database current.

Features
- Real-time hazard alerts — Notifies users of accident-prone areas along their route as they drive
- Crowdsourced reporting — Users can flag new hazards they encounter, keeping the database current instead of relying on static accident-zone data
- Emergency contacts — Quick access to emergency contact notifications triggered by reported incidents
- Route-aware alerting — Uses live geolocation and route matching to surface only relevant hazards along the user's path


| Layer | Technology |
|---|---|
| Mobile Client | React Native |
| Backend | Node.js |
| Database | MongoDB |

Note: Built as a university project (2022). Shared here for reference — not actively maintained.
